**DocuNest – Privacy Policy Repository**

This repository hosts the official Privacy Policy for the DocuNest mobile application.

DocuNest is a privacy-focused document organizer that allows users to securely store and manage personal documents directly on their device. The application is designed with a strong emphasis on offline functionality and local data storage.

 **About DocuNest**

DocuNest helps users organize important documents into profiles and bundles such as:

=> Personal documents

=> Family records

=> Health files

=> Insurance papers

=> Certificates and IDs

All data remains on the user’s device and is never transmitted to external servers.

 Privacy First Approach

DocuNest is built with privacy as a core principle:

 => No user tracking

 => No analytics collection

 => No cloud uploads

 => No third-party advertising SDKs

 => 100% offline operation
